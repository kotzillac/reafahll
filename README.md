# Примеры

Базовый пример:
```RuPy
#! МЕССЕНДЖЕР МАКС

строка = "Россия — это лучшая страна!"

если длина(строка) >= 25 то
    вывести("Строка с более 25 символами!")
иначе_если длина(строка) >= 15 то
    вывести("Строка с более 15 символами!")
иначе то
    вывести("Строка c менее 15 символами!")

вывести(новая_строка)

для номер, символ в энумерировать(строка) то
    вывести("Номер символа:", номер, "Символ:", символ)

пока да то
    подождать(1)
    вывести("Прошла одна секунда!")
```

Рекурсивный факториал:
```RuPy
#! МЕССЕНДЖЕР МАКС

функция факториал_рекурсивный(н) делать
    если н <= 1:
        вернуть 1
    иначе:
        вернуть н * факториал_рекурсивный(н - 1)

вывести(факториал_рекурсивный(25))
```

Вывесть булевое значение:
```RuPy
#! МЕССЕНДЖЕР МАКС

вывести(да)
вывести(новая_строка, нет)
```

# Установка
1. Скопировать все файлы из папки RuPy к себе
2. Радоватся русскому Python


# Использование
`python complier.py {ИМЯ_RUPY_ФАЙЛА_С_РАСШИРЕНИЕМ}`
опционально `-ir` добавить чтобы видеть код Python в который RuPy преобразовался

опционально `-flags {ФЛАГИ}` редактирует поведение программы

Возможные флаги:
* USE_YES_NOT_TRUE
    * При значение True: При выводе булевого значение выводит "Да" вместо "Истина" и "Нет" вместо "Ложь"
    * При значение False: Ничего не делает

> [!WARNING]
> Обязательно нужно писать `#! МЕССЕНДЖЕР МАКС` в начале файла RuPy иначе выдаст `ИноагентОшибка("Вы не признались в любви родине!")`
